# Orechou's Homebrew Tap

Personal Homebrew tap for macOS applications by @orechou.

## Installation

```bash
brew tap orechou/homebrew
```

## Available Formulae

| Formula | Description | Install |
|---------|-------------|---------|
| [nextor](./Formula/nextor.rb) | A modern Markdown editor with Mermaid support | `brew install nextor` |

## Usage

```bash
# Install
brew install nextor

# Upgrade
brew upgrade nextor

# Uninstall
brew uninstall nextor
```
